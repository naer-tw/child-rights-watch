🔍 六項跨公約結構性發現

1. 身障女性 在 CRPD 文件中隱形。 CRPD 0 段 對 CEDAW 190 段。違反 CEDAW GR-18 + CRPD §6。
→ CRPD PI-17 詳細頁
2. 家暴在 CRPD 缺席。 CRPD 1 段 對 CEDAW 459 / CRC 515。身障女性交織暴力無政府論述。
→ CRPD PI-17 詳細頁
3.「合理調整」悖論。 CRPD 招牌概念在 CRPD 文件出現 7 次,但 CEDAW 100 / CRC 173。
→ CRPD PI-12 評分
4. SOGIESC 100× 集中於 CEDAW。 LGBTI / 性別認同政府論述壓倒性集中在婦女文件,而非兒少 / 身障框架。
→ CRPD PI-18 詳細頁→ CRC PI-14 詳細頁
5. 身障原住民完全隱形。 三公約 合計 0 段 — 三重弱勢族群不被政府識別。
→ CRPD PI-19 詳細頁→ CEDAW PI-15 詳細頁
6. LGBTQI+ 兒少 三公約都低。 CEDAW 2 / CRC 1 / CRPD 0 段 — 敏感交織議題缺乏系統政府論述。
→ CRC PI-14 詳細頁→ CRPD PI-18 詳細頁

📋 證據強度(32 PI 五維評分)

每 PI 從 0-100 五維評分(passages 0-40 / events 0-20 / cosignatories 0-15 / orgs 0-15 / 年跨 0-10)。分布:

  • 7 A 級(≥70) — 證據強,可直接做媒體 / 公聽會 / 立委質詢
  • 20 B/C 級(30-69) — 中等,聚焦補強後可倡議
  • 5 D 級(<30) — 政府 / 社會「不被看見」議題(4 維行動清單見 D 級研究 TODO)

🎯 對 UN 治理機制的訴求

  1. CEDAW 委員會 — 在 List of Issues 要求台灣政府第 5 次國家報告(2026-27)新增身障女性專章,附性別×障礙細分數據。
  2. CRPD 委員會 — 在 List of Issues 要求第 3 次國家報告(預計 2027)新增身障女性專章,涵蓋家暴 / 性侵 / 就業 / 醫療統計。
  3. 跨委員會聯合 List of Issues — CEDAW + CRPD 在身障女性交織議題協調共同提問(CEDAW GR-18、CRPD §6、CEDAW GR-39)。
