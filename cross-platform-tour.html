📚 30 秒簡介

本平台由 AABE 國教行動聯盟整合台灣三個公民社會監督平台 — 婦女(CEDAW)/ 兒少(CRC)/ 身權(CRPD), 共用同一套 evidence_pipeline.py 開源管線。從政府官網爬取 2,223 份政府文件, 建立 102,728 段 FTS5 全文索引,自動為 32 個議題找關聯證據(共 1,266 連結)。

本平台價值不在「資料量」,而在 跨公約對照 — 揭露單一 NGO 永遠看不到的交織盲點(身障女性 / LGBTI 兒少 / 身障原住民等)。

🔄 推薦使用流程

  1. 第一步:訪 統合儀表板(數字總覽)
  2. 第二步:訪 交織議題(看公約之間盲點)
  3. 第三步:用 跨平台搜尋找你關心的議題
  4. 第四步:點到單一平台議題卡,看完整脈絡
  5. 第五步:採用 倡議簡報範本產出文件
  6. 下載資料:All JSON APIs at /api/manifest.json
  7. 追蹤更新:平台每月自動刷新,可訂閱 GitHub repo
