## 平台立場聲明

- **性教育主軸為健康教育**(依 UNESCO 架構「可依國情採行」),性平議題「融入而非侵入」
- **重啟青少年性健康計畫**(對應淋病 +7×、梅毒 +53×)
- **本土化性教育綱要**(不照搬國外模板)
- **AABE 為民間組織**(非社團法人);英文簡稱 AABE
