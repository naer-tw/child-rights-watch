## 處理原則

1. **不另建主議題**:近期的依托咪酯、實驗教育、剴剴案、性平公聽會等,雖然量大,**併入** 14 個主議題之子議題或案例,避免議題稀釋。
   - 例外:處境不利兒少族群(移民/無國籍、LGBTQI+)因 CRC2-CO 直接點名且既有 12 議題無對應主軸,於 2026-04-25 新增為 PI-13、PI-14。標準為「**CRC2-CO 直接段落 + 既有 12 議題覆蓋不足 + 影子報告監督正當性需求**」三者俱備。
2. **xlsx 仍是真實來源**:`國教盟倡議追蹤系統.xlsx`、`立法院倡議監督追蹤系統.xlsx` 在 MVP 階段繼續使用,內容**逐筆移轉**到 Notion `advocacy_action` 表後再廢止。
3. **音檔逐字稿優先轉**:訪談 m4a → 逐字稿 → 拆段 → 標籤。優先處理 [賴月蜜老師訪談](~/Desktop/E36A36394ED03C46602B493AE6046D9E8A1F3D5F_賴月蜜老師.m4a)(估計與 PI-04/05 高度相關)。
4. **GEO 轉換的 html 檔**(2026-04-12、04-15、04-17 系列)直接作為對外發布的成品,在平台上以 `media_coverage` 連結引用。
