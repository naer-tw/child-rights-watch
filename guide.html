PART 16 個常見倡議任務,推薦這樣用

每個情境寫的是「現實中你正在做什麼 → 平台幫你 30 秒搞定哪一步」。從這六張卡開始,大概可以涵蓋 80% 的日常。

📋
2 分鐘 · 拜會前

立委辦公室拜會 / 助理 brief

明天要見 OO 立委助理,談「兒少權法修法」。需要在路上 5 分鐘掌握:AABE 主立場、3 個必引數據、最強 ASK。

  1. 打開 倡議者快查卡,點對應議題(例 PI-12 替代照顧)
  2. 截圖或按「🖨 列印 / 存 PDF」帶在身上
  3. 記住 1 句立場 + 3 個 ASK = 結束
→ 開啟倡議者快查卡
🎤
30 秒 · 媒體採訪

記者突然來電要 quote

記者問:「你們對 OO 議題立場是什麼?有沒有可引用的數據?」需要立刻找到一句可登報的 quote。

  1. 開 全平台搜尋,輸入記者問的關鍵字(例:剴剴、體罰、自殺)
  2. 看「議題卡」+「AABE 倡議」結果
  3. 從快查卡的「3 個可引述名言」直接念給記者
→ 全平台搜尋
🏛
5 分鐘 · 公聽會

公聽會 5 分鐘發言

立法院明天公聽會排你發言 5 分鐘。需要結構:現況 → 證據 → AABE 主張 → ASK,還要有對方論點的回應準備。

  1. 開 議題立場頁 找對應 PI 的「論辯張力」
  2. 用 快查卡 的「對方論點 vs 我方回應」做防守
  3. 到 政策軌跡 找最近的政府行動 / 案件 timeline
→ 14 議題立場
📰
30 分鐘 · 寫稿

寫新聞稿 / 政策建議書

要產出一份 800 字新聞稿或 3 頁政策建議書,需要 ≥ 3 個來源(政府/學術/國際)、CRC 條文對應、AABE 歷史立場。

  1. 用 CRC 條文反向索引 確認對應條號
  2. 用 關鍵發現 抓平台級數據(議題負荷 / 採納率 / 惡化趨勢)
  3. 用 民間影子報告 補 NGO 觀點
  4. 引用就用 快查卡 的 quotable
→ 平台關鍵發現
🎓
1 小時 · 議題研究

研究員 / 學生做議題研究

第一次接觸 OO 兒少議題,需要從零開始:這議題在台灣怎麼回事?CRC 怎麼說?歷年政策怎麼走?

  1. 從 議題地圖 找對應 PI
  2. 進 議題詳情頁 讀完整 10 章節(成人版+兒少版)
  3. 用 政策軌跡 看 1984-2027 重大事件 timeline
  4. 跨週期比較看 CRC 1/2/3 屆審查進度
→ 議題地圖
🌐
2 分鐘 · 國際對接

國際審查 / 跨平台分析

要寫雙語 Alternative Report,或要看 CRC × CRPD × CEDAW 在 OO 議題上的交叉觀點。

  1. 用 三公約共構搜尋(右上角 🌐 按鈕)做跨公約查詢
  2. 從 影子報告 取 5 audience 版本(專業/媒體/兒少/立法/社會)
  3. 到 跨週期比較 看屆別進度
→ 三公約搜尋

PART 216 個模塊各做什麼?

紅頂 = 倡議者必用,金頂 = 內部工作工具,深藍頂 = 研究分析。每張卡告訴你「是什麼 / 何時用 / 點哪裡」。

★ 倡議者快查卡
必用
14 議題各一張 A4 1-pager,含 AABE 立場 + 3 必引數據 + 3 可引名言 + 對話劇本 + 3 秒 ASK。
何時用? 進立委辦公室、媒體採訪、公聽會發言、街頭倡議。可列印或存 PDF 隨身。
→ 開啟 14 卡 index
14 議題立場索引
必用
一頁看懂 AABE 在 14 個 CRC 兒少議題上的核心立場、訴求、論辯張力、民間夥伴 chorus。
何時用? 想確認「我們在這議題究竟主張什麼」、論辯時找對方論點對策、新聞稿引述官方立場。
→ 開啟立場索引
議題詳情頁(14 本)
必用
每個 PI 一頁深度檔:議題摘要(成人+兒少版)/ CRC 對應 / 政府回應 / 民間觀察 / 案例 / 政策建議 / 國際比較。
何時用? 第一次接觸這議題、寫研究報告需要全貌、要查 CRC 條文對應與五大修法工程進度。
→ 從議題地圖進入
政策軌跡時間軸
必用
316 事件五方並陳:CRC 結論性意見、政府 NAP、AABE 倡議、立法修法、重大案件。可依議題、類型、年份、關鍵字篩選。
何時用? 要整理「這議題十年怎麼走的」、想看政府最近做了什麼、找特定年份重大事件。
→ 開啟時間軸
全平台搜尋
必用
跨 9 種類型 / 70054 筆索引(議題、CRC 條文、CO、影子報告、官方文件、NAP、NHRC、法規、修法、69597 全文段落)。
何時用? 不確定資料在哪頁、記者突然問特定關鍵字(剴剴 / 自殺 / 體罰)、要找文件中某句話。
→ 開啟搜尋
平台關鍵發現
必用
基於全平台 754 索引、441 PDF、27 影子報告之關鍵洞察:議題負荷 Top 6 / NGO 活躍度 / 採納率最低 / 惡化趨勢。
何時用? 寫稿要有獨家數據、被問「平台到底發現什麼」、做媒體 release 需要平台級洞察。
→ 開啟關鍵發現
CRC 條文反向索引
必用
30 條 CRC 條文 → 對應到本平台議題、結論性意見、民間影子報告。從條號切入找資料。
何時用? 引用條文時要確認條號、寫 Alternative Report 要對齊 CRC 條文順序、查特定條文台灣現況。
→ 開啟條文索引
議題地圖
必用
14 PI 視覺化地圖,依 CRC cluster 分為 9 大群(一般原則 / 暴力 / 健康 / 教育 / 司法 / 數位...)。
何時用? 第一次來想全覽、要找跨議題關聯、做議題分類匯整。
→ 開啟議題地圖
CRC 官方文件
研究
110 份 CRC 官方 PDF + 1:1 markdown 轉檔,涵蓋國家報告、結論性意見、問題清單、議題討論。
何時用? 要直接讀政府原文、查特定段落引用出處、寫學術研究需要原始 PDF。
→ 官方文件清單
民間影子報告
研究
27 份民間 NGO 影子報告(15 份 CRC 官方蒐集 + 12 份 AABE 平台補建),涵蓋 28 個 NGO 觀點。
何時用? 要看其他 NGO 怎麼說同議題、做交叉比較、找 NGO 夥伴聯名連署。
→ 影子報告清單
NHRC 監測指標
研究
6 個國家人權委員會兒少監測指標儀表板,含趨勢符號(↑改善 / ─穩定 / ↓惡化)。
何時用? 要找官方統計數據、寫稿引用 NHRC、判斷議題是否惡化中。
→ NHRC 儀表板
NAP 追蹤
研究
人權國家行動計畫(NAP)兩期追蹤,含 AABE 對每項行動的執行進度評估。
何時用? 要批評政府承諾未做到、看哪些 CRC 結論性意見已被列入 NAP、追蹤期程。
→ NAP 追蹤
15 法規
研究
與兒少人權直接相關 15 個法規(兒少權法、少事法、家事法、性平法、保護令法...)+ 42 修法歷程。
何時用? 要查條文最新修法狀況、寫修法建議書、引用法規條號。
→ 法規清單
每週動態
研究
本週新增之 advocacy / timeline / NAP 變動摘要,可訂閱 markdown 版。
何時用? 每週一上班看上週發生什麼、做組織內部 weekly digest、追蹤政府最新動向。
→ 每週動態
跨週期比較
研究
CRC 1/2/3 屆審查屆別比較、NAP 第 1/2 期比較、議題進展軌跡。
何時用? 要回答「政府 5 年來究竟有沒有進步」、寫年度報告、做學術比較。
→ 跨週期比較
鏈式查詢
工具
14 PI 完整資料鏈展開:倡議 → CRC → CO → NAP → 法規 → 評估,六層串連。
何時用? 要做完整資料證據鏈展示、寫深度報告、培訓新進倡議者議題全貌。
→ 鏈式查詢
倡議追蹤矩陣
內部
內部工作區:管理 81 件 advocacy 之採納狀態、回應期限、跟進負責人。
何時用? AABE 內部追蹤倡議成果、定期 review、產製季報。(需登入 AABE 自家人帳號)
→ 倡議追蹤矩陣

🖨 列印 PDF 快查卡 / 議題詳情 / 立場頁都有 print CSS,Cmd+P 直接存 A4 PDF,排版自動最佳化。

🌐 三公約共構 右上角 🌐 三公約搜尋可跨 CRC + CRPD + CEDAW 三平台查詢同關鍵字。

📱 手機友善 全平台已優化手機版面(Wave 155),可直接用 Safari / Chrome 瀏覽。立委辦公室拜會時手機開議題詳情頁。

PART 5常見問題 FAQ

Q1. 我可以直接引用平台內容嗎?授權怎麼算?

本平台採 CC BY 4.0 授權,引用請標示來源「國教行動聯盟兒少人權監督平台」與取得日期。AABE 立場頁、快查卡、findings 都可直接引用;CRC 官方文件 PDF 屬政府公開文件,引用請註明出處頁碼。

Q2. 平台多久更新一次?資料新鮮嗎?

平台採每月維護(monthly_maintenance.sh 自動跑),包含:

  • 每月 1 號:政府網站 SSL+cookie+stable hash 監測,發現變更自動入庫
  • NAAES 政策站新文章自動抓取
  • 每月底:全部 SQLite 重建 + 14 PI 頁 + 政策軌跡 + 立場 + 快查卡同步

每頁右下角的「最後更新」日期是該頁實際生成日期。

Q3. 平台 SQLite 資料庫可以下載嗎?

可以,data/crw.db 是公開檔。包含 14 主要表(policy_issue / advocacy_action / concluding_observation / nap_action / domestic_law / law_amendment / crc_article / external_shadow_report / nhrc_indicator / issue_timeline / aabe_position 等)+ 7 關聯表。建議用 DB Browser for SQLite 開啟。

Q4. 我發現資料錯了 / 想補充,怎麼辦?

歡迎 email AABE 平台維護組,或在我們的 GitHub repo 開 issue。社區協作的具體流程:

  • 事實錯誤:附證據連結直接 email,我們會在下一次月維護更新
  • 新增倡議行動:NGO 團體發了新政策建議書 → 寄 markdown 或 PDF + JSON metadata,可進 advocacy_action 表
  • 議題範圍補充:大型結構性增補需經 AABE 政策室評估
Q5. 為什麼有些頁顯示「需登入(AABE 自家人)」?

本平台分兩層:公開內容(任何人可瀏覽)和 AABE 內部工作區(需登入)。內部區包含倡議追蹤矩陣、影子報告產生器、倡議工作底稿、內部自審等,因為是 AABE 政策室之工作中文件,品質尚未到對外公開水準。登入帳號 AABE / 密碼 1234(可在瀏覽器儲存)。

Q6. 能用手機看嗎?

可以。Wave 155 後全平台已 mobile responsive 優化,375 px 至 1280 px 各尺寸均可瀏覽。立委辦公室拜會、媒體採訪、街頭倡議建議直接用手機開「倡議者快查卡」對應 PI 頁,標題、3 必引數據、3 可引名言、3 秒 ASK 全部完整顯示。

Q7. 影子報告 5 個版本(專業 / 媒體 / 兒少 / 立法 / 社會)有什麼差別?

同一份證據材料,針對 5 種不同讀者寫不同版本:

  • 專業版:給律師、研究員、CRC 委員 — 完整論證 + 引註
  • 媒體版:給記者 — 重點化、quotable、含圖表
  • 兒少版:給 12 歲以下小朋友 — 用故事與比喻
  • 立法版:給立委 — 直指修法工程 + ASK
  • 社會版:一般大眾 — 平實口語

每月維護自動產出 5 版,在 shadow_report/build/ 目錄。

Q8. CRC × CRPD × CEDAW 三公約共構是什麼?

AABE 同時維護三個監督平台:本平台(CRC)、CRPD 監督平台(身障)、CEDAW 監督平台(婦女)。三公約有大量交叉議題(身障兒少、女性兒少、LGBTQI+ 等)。「三公約共構搜尋」(右上 🌐 按鈕)可跨三平台同時查詢關鍵字,適合做交叉議題分析或寫雙語綜合報告。

Q9. 我想引用平台某個段落,怎麼找正確的引用格式?

建議引用格式範例:

  • 議題卡:國教行動聯盟兒少人權監督平台,「PI-12 替代照顧與機構安置(剴剴案後)」,2026-05-06。https://naer-tw.github.io/child-rights-watch/issues/PI-12.html
  • AABE 立場:國教行動聯盟,「PI-12 政策立場」,引自《兒少人權監督平台 14 議題立場索引》(aabe_position 表),2026-05-06。
  • CRC 結論性意見:CRC 委員會,《結論性意見 第 2 屆審查》第 36 段(CRC2-CO36),2022。
